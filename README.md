### Work

**[Neural Holography Engine](https://github.com/gouravgangwardev/Neural-Holography-Engine)** — `Python` `Three.js` `NeRF` `OpenCV`  
AI pipeline converting 2D images into real-time interactive 3D holograms. Full-stack spatial computing: neural reconstruction (TripoSR, GET3D, NeRF), mesh processing, Three.js rendering, and sub-15ms gesture control via MediaPipe. References work from Stability AI, NVIDIA, DeepMind, and MIT.

**[Math Supernova Lab](https://github.com/gouravgangwardev/maths-supernova)** — `TypeScript` `Vite` `Bun` `Canvas`  
High-performance computation engine for real-time mathematical visualization in the browser. Modular, memoized, fully type-safe. Built to handle complex function rendering with zero perceptible lag.

**[Trading Backtest Engine](https://github.com/gouravgangwardev/trading-backtest-engine)** — `Python` `Quant Finance`  
Modular backtesting framework for algorithmic trading strategies. Designed for extensibility — swap strategies, data sources, and metrics without rewriting core logic.

**[Chemical Process Library](https://github.com/gouravgangwardev/ChemicalProcessLibrary-OpenModelica)** — `OpenModelica` `Scientific Computing`  
Reusable simulation library with plug-and-play reactors and replaceable reaction kinetics. Built for validated, production-grade process simulation.
