### ◈ Neural Holography Engine &nbsp;&nbsp; [`Python` `JS` `CV` `NeRF`](https://github.com/gouravgangwardev/Neural-Holography-Engine)

```
2D Image → Neural 3D Reconstruction → Real-Time Holographic Projection → Gesture Control
```

A unified AI framework that converts any 2D image into an interactive, gesture-controlled 3D hologram. Not a demo — a full-stack spatial computing pipeline referencing peer-reviewed research from **Stability AI**, **NVIDIA**, **Google DeepMind**, **MIT**, and **UC Berkeley**.

Architecture spans 6 independent layers: input processing (OpenCV), neural reconstruction (TripoSR / GET3D / NeRF / CRM), mesh optimization (Blender API), real-time rendering (Three.js), gesture interaction (MediaPipe / Leap Motion), and physical projection (Pepper's Ghost / AR Glass).

```
Medical accuracy:  98% (CRM / MeshHeart)
Gesture latency:   <15ms (MediaPipe)
3D inference:       0.5s (TripoSR)
```

→ **[View Repository](https://github.com/gouravgangwardev/Neural-Holography-Engine)**

### ◈ Math Supernova Lab &nbsp;&nbsp; [`TypeScript` `Vite` `Bun` `Canvas`](https://github.com/gouravgangwardev/maths-supernova)

```
Expression Input → Parser → Computation Engine → Real-Time Visualization
```

A high-performance mathematics engine built for real-time computation and interactive graph rendering in the browser. Modular architecture — each mathematical operation is an isolated, composable unit. Built with lazy loading, memoization, and static TypeScript throughout.

Stack: **TypeScript + Vite + Tailwind + Bun** — blazing-fast build pipeline with zero-compromise type safety.

→ **[View Repository](https://github.com/gouravgangwardev/maths-supernova)**
