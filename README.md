## 🚀 Featured Projects

### 🧠 [Neural Holography Engine](https://github.com/gouravgangwardev/Neural-Holography-Engine)
> *AI-Driven Spatial Reconstruction & Real-Time Volumetric Projection*

Converts any 2D image into an interactive 3D hologram — fusing neural 3D reconstruction (TripoSR, NeRF, GET3D), real-time rendering via Three.js, and gesture-based interaction via MediaPipe. Architecture spans 6 layers from input to physical projection. Built on peer-reviewed research from Stability AI, NVIDIA, DeepMind, and MIT.

`Python` `JavaScript` `Three.js` `OpenCV` `NeRF` `MediaPipe`

---

### 📐 [Math Supernova Lab](https://github.com/gouravgangwardev/maths-supernova)
> *High-Performance Mathematics Engine with Real-Time Visualization*

A modular computation engine that parses, evaluates, and visualizes complex mathematical functions in the browser — live, with zero lag. Built with Canvas/SVG rendering, memoization, and lazy-loading. Stack: TypeScript + Vite + Tailwind + Bun.

`TypeScript` `Vite` `Tailwind CSS` `Bun` `Canvas API`

---

### 📈 [Trading Backtest Engine](https://github.com/gouravgangwardev/trading-backtest-engine)
> *Algorithmic Strategy Testing Framework*

A quantitative backtesting framework for evaluating trading strategies on historical market data. Designed for modularity — swap strategies, data sources, and performance metrics without rewriting the core engine.

`Python` `Algorithmic Trading` `Quant Finance`

---

### ⚗️ [Chemical Process Library — OpenModelica](https://github.com/gouravgangwardev/ChemicalProcessLibrary-OpenModelica)
> *Modular Chemical Process Simulation*

A reusable simulation library in OpenModelica featuring plug-and-play reactors, replaceable reaction kinetics, and validated example scenarios. Demonstrates deep command over scientific computing and domain modelling.

`OpenModelica` `Chemical Engineering` `Simulation`

---

### 🔐 [Cowrie SSH Honeypot](https://github.com/gouravgangwardev/cowrie)
> *Cybersecurity Research — SSH/Telnet Threat Intelligence*

Forked and extended Cowrie honeypot for capturing attacker behaviour patterns. Demonstrates active interest in cybersecurity infrastructure and threat analysis.
